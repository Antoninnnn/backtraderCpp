### Installation(Unix)

git项目地址：https://github.com/Antoninnnn/backtraderCpp

一些可能会在不同系统中出现的问题：

库名在不同系统中以及不同包管理器中有细微差别，可能会报错

库本身不完整需要重新编译

路径链接问题

静态与动态库问题



##### MacOS

`brew install ta-lib`

`brew install boost`

`brew install curl`

完成安装这三个库后通过查找brew的默认路径，在笔者处头文件位置为

 ` /usr/local/Cellar/boost/1.75.0_1/include`

  `/usr/local/Cellar/ta-lib/0.4.0/include/ta-lib`

   ` /usr/local/Cellar/curl/7.75.0/include/curl`·，

进入CMakeLists.txt修改Unix中的include_directories。

同样的在link_directories中修改编译生成的库的路径。

最后检查CMakeLists.txt中target_link_libraries里库的名称是否有误。



##### Ubuntu(Linux)

`pip install TA-lib`

`pip install boost`

`pip install boost`

类似于MacOS系统，找到这些库的位置后进入CMakeLists.txt修改unix的include_directories与link_directories。

最后检查CMakeLists.txt中target_link_libraries里库的名称是否有误。
